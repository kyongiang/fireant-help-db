# CỘNG ĐỒNG

Cộng đồng là nơi người dùng chia sẻ thông tin và trao đổi kiến thức

![](https://help.fireant.vn/files/2FXCtdJTpK1YPDlnMuAQ)

Trong chức năng cộng đồng bạn có các lựa chọn sau

* Xem các **Bài viết từ cộng đồng**
* Xem hồ sơ của các **Thành viên** có nhiều đóng góp cho cộng đồng
* Tạo các **Hội nhóm** có cùng sở thích hay mối quan tâm để chia sẻ thông tin
