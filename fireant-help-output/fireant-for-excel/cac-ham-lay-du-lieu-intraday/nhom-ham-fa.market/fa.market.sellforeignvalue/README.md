# FA.Market.SellForeignValue

## Cú pháp

```
=FA.Market.SellForeignValue("Mã sàn")
```

**Trong đó:**

* **Mã sàn:** Mã sàn cần lấy thông tin. Mã sàn có thể lấy các giá trị HSX, HNX, UPCOM

## Giá trị trả về

Tổng giá trị nhà đầu tư nước ngoài bán trên sàn
