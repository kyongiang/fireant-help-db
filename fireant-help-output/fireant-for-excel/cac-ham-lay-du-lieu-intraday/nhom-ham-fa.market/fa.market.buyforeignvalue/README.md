# FA.Market.BuyForeignValue

## Cú pháp

```
=FA.Market.BuyForeignValue("Mã sàn")
```

**Trong đó:**

* **Mã sàn:** Mã sàn cần lấy thông tin. Mã sàn có thể lấy các giá trị HSX, HNX, UPCOM

## Giá trị trả về

Tổng giá trị nhà đầu tư nước ngoài mua trên sàn
