# FA.Market.PTValue

## Cú pháp

```
=FA.Market.PTValue("Mã sàn")
```

**Trong đó:**

* **Mã sàn:** Mã sàn cần lấy thông tin. Mã sàn có thể lấy các giá trị HSX, HNX, UPCOM

## Giá trị trả về

Tổng giá trị giao dịch thỏa thuận trên sàn
