# FA.Market.SellForeignQuantity

## Cú pháp

```
=FA.Market.SellForeignQuantity("Mã sàn")
```

**Trong đó:**

* **Mã sàn:** Mã sàn cần lấy thông tin. Mã sàn có thể lấy các giá trị HSX, HNX, UPCOM

## Giá trị trả về

Tổng khối lượng nhà đầu tư nước ngoài bán trên sàn
