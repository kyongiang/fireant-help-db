# FA.Market.TotalTrade

## Cú pháp

```
=FA.Market.TotalTrade("Mã sàn")
```

**Trong đó:**

* **Mã sàn:** Mã sàn cần lấy thông tin. Mã sàn có thể lấy các giá trị HSX, HNX, UPCOM

## Giá trị trả về

Tổng số lệnh khớp trên sàn
