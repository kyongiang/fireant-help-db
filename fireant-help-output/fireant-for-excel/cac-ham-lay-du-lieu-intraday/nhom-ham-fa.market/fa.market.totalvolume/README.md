# FA.Market.TotalVolume

## Cú pháp

```
=FA.Market.TotalVolume("Mã sàn")
```

**Trong đó:**

* **Mã sàn:** Mã sàn cần lấy thông tin. Mã sàn có thể lấy các giá trị HSX, HNX, UPCOM

## Giá trị trả về

Tổng khối lượng khớp lệnh trên sàn
