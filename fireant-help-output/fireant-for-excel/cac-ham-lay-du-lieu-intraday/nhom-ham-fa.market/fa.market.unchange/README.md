# FA.Market.Unchange

## Cú pháp

```
=FA.Market.Unchange("Mã sàn")
```

**Trong đó:**

* **Mã sàn:** Mã sàn cần lấy thông tin. Mã sàn có thể lấy các giá trị HSX, HNX, UPCOM

## Giá trị trả về

Số lượng mã không thay đổi giá trên sàn
