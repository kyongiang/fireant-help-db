# FA.Market.PTVolume

## Cú pháp

```
=FA.Market.PTVolume("Mã sàn")
```

**Trong đó:**

* **Mã sàn:** Mã sàn cần lấy thông tin. Mã sàn có thể lấy các giá trị HSX, HNX, UPCOM

## Giá trị trả về

Tổng khối lượng giao dịch thỏa thuận trên sàn
