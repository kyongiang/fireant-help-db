# FA.Market.TotalValue

## Cú pháp

```
=FA.Market.TotalValue("Mã sàn")
```

**Trong đó:**

* **Mã sàn:** Mã sàn cần lấy thông tin. Mã sàn có thể lấy các giá trị HSX, HNX, UPCOM

## Giá trị trả về

Tổng giá trị khớp lệnh trên sàn
