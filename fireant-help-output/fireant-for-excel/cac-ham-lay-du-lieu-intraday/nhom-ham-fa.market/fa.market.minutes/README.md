# FA.Market.Minutes

## Cú pháp

```
=FA.Market.Minutes("Mã sàn")
```

**Trong đó:**

* **Mã sàn:** Mã sàn cần lấy thông tin. Mã sàn có thể lấy các giá trị HSX, HNX, UPCOM

## Giá trị trả về

Số phút thực sự giao dịch của thị trường (bỏ qua thời gian nghỉ)
