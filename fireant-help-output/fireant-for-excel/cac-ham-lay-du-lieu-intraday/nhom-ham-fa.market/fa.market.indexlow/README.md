# FA.Market.IndexLow

## Cú pháp

```
=FA.Market.IndexLow("Mã index")
```

**Trong đó:**

* **Mã index:** Mã index cần lấy thông tin. Mã index có thể lấy các giá trị VNINDEX, HNXINDEX, UPINDEX, VN30 và HNX30

## Giá trị trả về

Giá trị index thấp nhất trong ngày
