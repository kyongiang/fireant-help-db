# FA.Market.Untraded

## Cú pháp

```
=FA.Market.Untraded("Mã sàn")
```

**Trong đó:**

* **Mã sàn:** Mã sàn cần lấy thông tin. Mã sàn có thể lấy các giá trị HSX, HNX, UPCOM

## Giá trị trả về

Số lượng mã không có giao dịch trên sàn
