# FA.Market.IndexBasic

## Cú pháp

```
=FA.Market.IndexBasic("Mã index")
```

**Trong đó:**

* **Mã index:** Mã index cần lấy thông tin. Mã index có thể lấy các giá trị VNINDEX, HNXINDEX, UPINDEX, VN30 và HNX30

## Giá trị trả về

Giá trị index tham chiếu (Index đóng cửa phiên trước)
