# FA.Market.Advances

## Cú pháp

```
=FA.Market.Advances("Mã sàn")
```

**Trong đó:**

* **Mã sàn:** Mã sàn cần lấy thông tin. Mã sàn có thể lấy các giá trị HSX, HNX, UPCOM

## Giá trị trả về

Số lượng mã tăng giá trên sàn
