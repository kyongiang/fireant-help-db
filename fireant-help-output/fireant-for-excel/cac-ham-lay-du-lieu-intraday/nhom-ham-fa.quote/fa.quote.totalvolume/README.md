# FA.Quote.TotalVolume

## Cú pháp

```
=FA.Quote.TotalVolume("Mã CK")
```

**Trong đó:**

* **Mã CK:** Mã chứng khoán của cổ phiếu cần lấy thông tin

## Giá trị trả về

Tổng khối lượng khớp trong phiên hiện tại của mã chứng khoán
