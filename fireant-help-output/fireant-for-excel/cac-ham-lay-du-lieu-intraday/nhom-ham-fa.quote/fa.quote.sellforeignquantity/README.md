# FA.Quote.SellForeignQuantity

## Cú pháp

```
=FA.Quote.SellForeignQuantity("Mã CK")
```

**Trong đó:**

* **Mã CK:** Mã chứng khoán của cổ phiếu cần lấy thông tin

## Giá trị trả về

Khối lượng bán ra của nhà đầu tư nước ngoài với mã chứng khoán
