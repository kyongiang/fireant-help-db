# FA.Quote.BuyCount

## Cú pháp

```excel-formula
=FA.Quote.BuyCount("Mã CK")
```

**Trong đó:**

* **Mã CK:** Mã chứng khoán của cổ phiếu cần lấy thông tin

## Giá trị trả về

Số lệnh đặt mua của mã chứng khoán trong phiên
