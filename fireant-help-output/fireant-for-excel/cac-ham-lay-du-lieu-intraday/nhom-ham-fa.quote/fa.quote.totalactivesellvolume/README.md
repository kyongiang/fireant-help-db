# FA.Quote.TotalActiveSellVolume

## Cú pháp

```
=FA.Quote.TotalActiveSellVolume("Mã CK")
```

**Trong đó:**

* **Mã CK:** Mã chứng khoán của cổ phiếu cần lấy thông tin

## Giá trị trả về

Tổng khối lượng bán chủ động trong phiên của mã chứng khoán tính đế thời điểm hiện tại
