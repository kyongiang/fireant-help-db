# FA.Quote.PriceHigh

## Cú pháp

```
=FA.Quote.PriceHigh("Mã CK")
```

**Trong đó:**

* **Mã CK:** Mã chứng khoán của cổ phiếu cần lấy thông tin

## Giá trị trả về

Giá khớp lệnh cao nhất của mã chứng khoán trong phiên hiện tại
