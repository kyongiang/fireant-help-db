# FA.Quote.Time

## Cú pháp

```
=FA.Quote.Time("Mã CK")
```

**Trong đó:**

* **Mã CK:** Mã chứng khoán của cổ phiếu cần lấy thông tin

## Giá trị trả về

Thời gian giá của chứng khoán được cập nhật lần cuối trong phiên
