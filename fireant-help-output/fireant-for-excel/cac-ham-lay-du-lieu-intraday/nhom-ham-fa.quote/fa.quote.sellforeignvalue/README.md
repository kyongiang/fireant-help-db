# FA.Quote.SellForeignValue

## Cú pháp

```
=FA.Quote.SellForeignValue("Mã CK")
```

**Trong đó:**

* **Mã CK:** Mã chứng khoán của cổ phiếu cần lấy thông tin

## Giá trị trả về

Giá trị bán ra của nhà đầu tư nước ngoài với mã chứng khoán
