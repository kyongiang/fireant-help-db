# FA.Quote.TotalActiveBuyVolume

## Cú pháp

```
=FA.Quote.TotalActiveBuyVolume("Mã CK")
```

**Trong đó:**

* **Mã CK:** Mã chứng khoán của cổ phiếu cần lấy thông tin

## Giá trị trả về

Tổng khối lượng mua chủ động trong phiên của mã chứng khoán tính đế thời điểm hiện tại
