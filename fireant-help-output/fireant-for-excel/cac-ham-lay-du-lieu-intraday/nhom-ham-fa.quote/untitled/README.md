# FA.Quote.QuantityAsk3

## Cú pháp

```
=FA.Quote.QuantityAsk3("Mã CK")
```

**Trong đó:**

* **Mã CK:** Mã chứng khoán của cổ phiếu cần lấy thông tin

## Giá trị trả về

Khối lượng đặt bán ở mã chứng khoán tương ứng giá đặt bán thấp thứ ba
