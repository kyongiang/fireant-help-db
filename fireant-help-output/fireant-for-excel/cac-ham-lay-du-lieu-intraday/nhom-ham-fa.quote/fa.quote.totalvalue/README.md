# FA.Quote.TotalValue

## Cú pháp

```
=FA.Quote.TotalValue("Mã CK")
```

**Trong đó:**

* **Mã CK:** Mã chứng khoán của cổ phiếu cần lấy thông tin

## Giá trị trả về

Tổng giá trị khớp trong phiên hiện tại của mã chứng khoán
