# FA.Quote.PriceLast

## Cú pháp

```
=FA.Quote.PriceLast("Mã CK")
```

**Trong đó:**

* **Mã CK:** Mã chứng khoán của cổ phiếu cần lấy thông tin

## Giá trị trả về

Giá khớp lệnh gần nhất của mã chứng khoán ở phiên hiện tại. Trả về giá tham chiếu nếu phiên hiện tại chưa có khớp lệnh.
