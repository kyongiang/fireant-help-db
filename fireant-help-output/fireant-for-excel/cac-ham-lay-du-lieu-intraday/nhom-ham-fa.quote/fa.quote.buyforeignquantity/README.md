# FA.Quote.BuyForeignQuantity

## Cú pháp

```excel-formula
=FA.Quote.BuyForeignQuantity("Mã CK")
```

**Trong đó:**

* **Mã CK:** Mã chứng khoán của cổ phiếu cần lấy thông tin

## Giá trị trả về

Khối lượng mua vào của nhà đầu tư nước ngoài với mã chứng khoán
