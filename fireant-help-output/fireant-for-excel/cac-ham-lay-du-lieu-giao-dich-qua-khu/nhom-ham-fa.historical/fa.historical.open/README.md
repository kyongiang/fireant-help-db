# FA.Historical.Open

## Cú pháp

```
=FA.Historical.Open("Mã CK", "Ngày bắt đầu", "Ngày kết thúc")
```

Hàm FA.Historical.Open có các tham số sau:

* **Mã CK:** Mã chứng khoán cần lấy thông tin hoặc index (VNINDEX, HNXINDEX, UPINDEX, VN30, HNX30,...)&#x20;
* **Ngày bắt đầu**: Ngày bắt đầu lấy dữ liệu, định dạng "yyyy-MM-dd"
* **Ngày kết thúc:** Ngày kết thúc lấy dữ liệu, định dạng "yyyy-MM-dd"

## Giá trị trả về

Một chuỗi các giá trị trong đó mỗi phần tử là giá mở cửa ứng với 1 ngày giao dịch trong khoảng thời gian
