# FA.BalanceSheet.Reports

Cú pháp

FA.BalanceSheet.Reports(Mã chứng khoán, Năm, Quý, Số lượng, Đơn vị)

Hàm FA.BalanceSheet.Reports có các tham số sau :

* Mã chứng khoán: Mã chứng khoán của cổ phiếu cần lấy thông tin&#x20;
* Năm: Năm cuối cùng lấy báo cáo&#x20;
* Quý: Quý cuối cùng lấy báo cáo. Đặt bằng 0 nếu muốn lấy báo cáo theo năm&#x20;
* Số lượng: Số lượng báo cáo lấy về&#x20;
* Đơn vị: Đơn vị hiển thị số liệu
