# FA.Stock.StockSymbolsByExchange

## Cú pháp

```
=FA.Stock.StockSymbolsByExchange("Tên sàn")
```

**Trong đó:**

* **Tên sàn**: Tên sàn giao dịch của mã chứng khoán muốn lấy thông tin (HSX, HNX, UPCOM)

## Giá trị trả về <a href="#gia-tri-tra-ve" id="gia-tri-tra-ve"></a>

Danh sách mã chứng khoán niêm yết trên sàn giao dịch
