# Nhóm hàm FA.Settings

Hiện tại nhóm hàm này chỉ có duy nhất 1 hàm xác định ngôn ngữ sử dụng để hiển thị dữ liệu.

## **Cú pháp**

Để biết ngôn ngữ đang sử dụng hiển thị dữ liệu là ngôn ngữ nào, nhập công thức sau vào một ô mà bạn dự định hiện thị kết quả:

```
=FA.Settings.Language()
```

**Kết quả**:

vi-vn

Kết quả trên cho biết ngôn ngữ đang sử dụng là tiếng Việt
