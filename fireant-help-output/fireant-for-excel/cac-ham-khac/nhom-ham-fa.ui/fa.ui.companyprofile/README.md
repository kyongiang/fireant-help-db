# FA.UI.CompanyProfile

## Cú pháp

```
=FA.UI.CompanyProfile("Mã CK", ["Tiêu đề"])
```

**Trong đó:**

* **Mã CK:** Mã chứng khoán của cổ phiếu cần xem hồ sơ&#x20;
* **Tiêu đề:** Tiêu đề của liên kết dùng để mở hồ sơ. Nếu không truyền tham số này thì hàm sẽ dùng giá trị

  mặc định là 'Hồ sơ'

## Giá trị trả về

Giá trị trả về là một liên kết. Khi bấm vào liên kết này, hồ sơ doanh nghiệp cho mã tương ứng sẽ được nạp vào một cửa sổ mới.
