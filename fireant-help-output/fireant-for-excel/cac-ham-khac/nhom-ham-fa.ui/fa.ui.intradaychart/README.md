# FA.UI.IntradayChart

## Cú pháp

```
=FA.UI.IntradayChart("Mã CK", ["Tiêu đề"])
```

**Trong đó:**

* **Mã CK**: Mã chứng khoán của cổ phiếu cần xem biểu đồ&#x20;
* **Tiêu đề:** Tiêu đề của liên kết dùng để mở biểu đồ. Nếu không truyền tham số này thì hàm sẽ dùng giá trị mặc định là **"Biểu đồ Intraday"**

## Giá trị trả về

Giá trị trả về là một liên kết. Khi bấm vào liên kết này, biểu đồ Intraday cho mã tương ứng sẽ được nạp vào một cửa sổ mới.
