# Cấu trúc fireant-help-output

> Tài liệu này mô tả bộ dữ liệu help FireAnt đã scrape offline từ [help.fireant.vn](https://help.fireant.vn) (GitBook).
> Dùng kèm thư mục `fireant-help-output` khi đưa vào RAG, chatbot hoặc AI khác.

## Mục đích

Đây là **mirror offline** toàn bộ trang trợ giúp FireAnt. Mỗi trang web tương ứng một thư mục chứa `README.md` (Markdown) và có thể có thư mục `images/` chứa ảnh minh họa đã tải về.

**Thống kê:** 428 trang · 251 ảnh · 9 nhánh gốc

## Quy ước file

| Thành phần | Mô tả |
|------------|--------|
| `README.md` | Nội dung bài viết (Markdown). Có frontmatter YAML: `title`, `source`. |
| `_meta.json` | Danh sách con trực tiếp của thư mục: `slug → { "title": "..." }`. Thứ tự key = thứ tự sidebar khi scrape. |
| `images/` | Ảnh cục bộ, tham chiếu trong MD dạng `![](images/image-1.png)`. |

## Cách tra cứu

**URL gốc → file local:**

```
https://help.fireant.vn/{đường-dẫn-url}
  → fireant-help-output/{đường-dẫn-url}/README.md
```

**Ví dụ:**

| URL | File local |
|-----|------------|
| `https://help.fireant.vn` | `index/README.md` |
| `https://help.fireant.vn/fireant-for-web/bo-cuc-layout/quan-ly-khong-gian-lam-viec` | `fireant-for-web/bo-cuc-layout/quan-ly-khong-gian-lam-viec/README.md` |
| `https://help.fireant.vn/fireant-for-excel/.../fa.quote.volume` | `fireant-for-excel/.../fa.quote.volume/README.md` |

Đọc trường `source` trong frontmatter của bất kỳ `README.md` nào để lấy URL gốc chính xác.

## Ghi chú kỹ thuật

- **`fireant-for-web-1`**: Nhánh riêng cho thư viện chỉ số kỹ thuật/cơ bản (42 trang), tách khỏi `fireant-for-web` trên site GitBook gốc.
- **Slug `untitled`, `untitled-1`, …**: Trang GitBook không có slug chuẩn (thường gặp trong tài liệu hàm Excel). Tra theo `title` trong `_meta.json` hoặc frontmatter.
- **`_meta.json`**: Chỉ liệt kê con **trực tiếp**; không phải mọi thư mục đều có file này.
- **Ảnh**: Một số `README.md` vẫn còn `<img src="https://help.fireant.vn/files/...">` song song với `images/image-N.png`.
- **`fireant-help/`**: Chỉ 1 trang, xuất hiện trong root `_meta.json` nhưng ít liên quan cây chính.
- **Node `orphan`**: Thư mục có `README.md` nhưng không có trong `_meta.json` của thư mục cha (xem `manifest.json`).

- **Cây đầy đủ (máy đọc):** [`manifest.json`](manifest.json) — mỗi node có `slug`, `title`, `path`, `source`, `has_readme`, `children`.
- **Nguồn scrape:** [`scraper.py`](../scraper.py) trong repo gốc.
- **Tái tạo manifest:** `python generate_structure_manifest.py`
